﻿#NameNPC

NameNPC is a addon that allows your players to goto an NPC and change their name to whatever they want.

##Features
- Spawnable NPC
- Dark VGUI Theme
- Blocked Words
- Price
- Language File
- Easy Config
- Custom Chat Messages, or use Notifications
